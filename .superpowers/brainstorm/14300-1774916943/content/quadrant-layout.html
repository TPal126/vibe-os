Quadrant Assignments

↖ Top Left — Claude Chat

Session tabs, conversation with rich cards, message input. The primary interaction surface.

↗ Top Right — Architecture / Visuals

Live architecture diagram (default), full knowledge graph, dev server preview. The "what does my system look like" pane.

↙ Bottom Left — Context Controls

Skills (checkbox toggle), repos, token budgets, file tree. The "what's in my context" pane. Toggle skills in/out, see token impact instantly.

↘ Bottom Right — Activity / Deep Dive

Audit log, decision history, console, event stream. The "what happened and why" pane. Flexible — put whatever you need here.

Key Interactions

  • Drag dividers to resize any quadrant — double-click to reset to 50/50
  • Double-click a pane's tab bar to maximize it full-screen (press Esc to restore)
  • Architecture diagram updates live as Claude edits files — nodes pulse when touched
  • Click a node in the architecture diagram to see its connections, or to filter the audit log to that module
  • Checkbox a skill to instantly add/remove it from the active context — token count updates in real-time
  • Session tabs in chat pane — switching sessions updates the architecture view to highlight that session's active repos
